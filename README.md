### Deployment

1. Create a virtual environment and install the Python libraries

```sh
conda create --name HW1 python=3.11 -y
conda activate HW1
```

2.Install PyQt

```sh
pip install PyQt6
```

3. Run GUI

```sh
python app.py
```
